Leaving One-time Setup

You have left the one-time setup process without configuring WiFi network access. To return to the one-time setup process, follow the steps below:
  1. Press the 'PWR' button for more than 2 seconds (NOTE: Pressing 'PWR' for more than 7 seconds will switch off power to the board)
  2. Reconnect this machine to the 'params_ap' network
  3. Visit 'http://params_hostname' in your browser
Alternatively, you can connect to the device via its serial port and run configure_edison --wifi on the command line. To learn about other configuration options, run configure_edison --help.
