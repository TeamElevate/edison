Set Device Password
Your device is currently not configured with a password. This password will be used for both the access point and for root login (must be between 8 and 63 characters long.) Please be aware that if 'Caps Lock' is "on", the letters of your password will be capitalized.

A root password must be set to enable SSH access to the device via wireless. Once set, it can be changed at a later time by running configure_edison --password on the command line.
